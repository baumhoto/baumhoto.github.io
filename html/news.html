  • Ok still figuring out how i will do multiple updates during a day.. propably will just add it at the top...

  • AppleInsider has collected some data on Macbook Pro 2016+ repair rates which states that they have increased with the redesigned models. I don't own one but there is rarely a day where at there at least is one post complaining about it. Most complains are about the new keyboard which seems to have issues with keys stop working. Interestingly there hasn't been a replacement program for it, but for the batteries in the MBP 13 2016.

    I still own a 13 inch MBP 2013, but i also own a 12 inch Macbook 2015, which was the first Macbook with the new shallow Butterfly keys. I haven't had any issues with the keys, but i only use it as a hobby notebook.

    Currently i would not buy an Macbook. First of all because they still use the last generation intel core cpu's and at least for the i3 and i5 versions the number of cores has duplicated (4 instead of 2) with the latest generation. Not sure about i7 version because they already had 4 cores afaik. Second i like to be able to install Linux natively via Dual boot and even after 1 1/2 years there is still stuff that is not working. TouchId will never work but sound and hibernate should be. Third i think the touchbar is not useful. Main reason is because Apple limited it to only a few models (MBP 13 and 15) and all others Macbook, IMac don't get it. This is similar to 3D-Toch on Iphones which shares the same fate it is not available on all new IOS devices event though it has been introduced almost 3 years ago.

    So currently if you want to buy and Macbook Pro without the Touchbar you are forced to buy the entry-level macbook pro 13 inch. But as a programmer i want a beefier machine. Fortunately i don't need a new notebook right now, if i need one it would be a tough decision. Dell XPS 13, Lenovo Carbon X1 or Macbook Pro...
